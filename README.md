## 常见问题 FAQ

**Q：bwh1 是搬瓦工官网吗，安全吗？**

bwh1.net 是搬瓦工官方推出的国内镜像站，和 bandwagonhost.com 数据完全同步。你在 bwh1.net 注册的账号，在 bwh81.net 或者 bandwagonhost.com 里都能直接登录。安全性和主站一样，是官方认证的域名。

**Q：搬瓦工支付宝能用吗？**

可以。搬瓦工支持支付宝直接扫码付款，结算时选择 Alipay 即可，没有额外手续费。此外还支持 PayPal、信用卡和银联。

**Q：买了不满意可以退款吗？**

新账号首次购买，30 天内不满意可以申请全额退款。退款在 KiwiVM 后台或者提工单都可以申请，一般会自动处理，不需要跟客服扯皮。

**Q：CN2 GIA-E 和 KVM 套餐差在哪里？**

核心差距是线路。KVM 套餐走的是普通 163 骨干线路，晚高峰可能会有拥堵和延迟波动。CN2 GIA-E 走电信 CN2 GIA 优化线路，晚高峰稳定性明显好一些，可选机房也多很多（12 个以上 vs KVM 的几个普通机房）。价格差了 3 倍多，体验差距也基本对得上这个倍数。

**Q：哪个机房速度最快？**

香港 CN2 GIA（HKHK\_8）是搬瓦工目前延迟最低的机房，但价格也最贵。对大多数用户来说，CN2 GIA-E 套餐下的 DC6 洛杉矶（电信）或日本大阪软银（联通）已经够快了，性价比好得多。
